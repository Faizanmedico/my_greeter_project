# Personalized Greeter Project

A simple Python application that demonstrates a basic project structure using a `src` directory.
It reads names from a `data/names.txt` file and prints personalized greetings.

## Project Structure

my_greeter_project/
├── .venv/
├── src/
│   ├── init.py
│   ├── main.py
│   └── utils.py
├── data/
│   └── names.txt
├── requirements.txt
└── README.md
